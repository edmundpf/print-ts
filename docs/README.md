# print-ts
> Basic print tools w/ time logging

## Install
``` bash
$ npm i -S @edmundpf/print-ts
```

## Usage
``` javascript
// Javascript
const Print = require('@edmundpf/print-ts').default
// Typescript
import Print from '@edmundpf/print-ts'
// Initialize and Run
const print = new Print()
print.success('Using print-ts!')
```
