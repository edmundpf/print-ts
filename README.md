# print-ts
> Basic print tools w/ time logging

## Install
``` bash
$ npm i -S print-ts
```

## Usage
``` javascript
// Javascript
const Print = require('print-ts').default
// Typescript
import Print from 'print-ts'
// Initialize and Run
const print = new Print()
print.success('Using print-ts!')
```
